Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

Markdown 语言在 2004 由John Gruber创建。

Markdown 编写的文档可以导出 HTML等多种格式的文档。

Markdown 编写的文档后缀为 .md和 .markdown。
